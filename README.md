# Estacionamento_motofrete
Agente Inteligente de Logística Urbana — Motofrete BH  Projeto de IA Generativa que utiliza dados públicos da BHTrans sobre estacionamentos destinados ao motofrete em Belo Horizonte para criar um agente virtual capaz de analisar mobilidade urbana, identificar padrões de uso e gerar insights inteligentes para apoio à logística urbana e tomada de decisão.  O agente integra dados reais históricos, processamento em Python e modelos de IA via API, permitindo consultas em linguagem natural sobre disponibilidade, distribuição e evolução das vagas de motofrete na cidade.
